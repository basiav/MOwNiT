add_points <- function(df) {
  if (identical(df,size_and_better_mult)){
      geom_point(aes(x = res_grouped$size, y = res_grouped$better_mult_mean, colour="better_mult"), data = df)
  }
    else {
        geom_point(aes(x = res_grouped$size, y = res_grouped$BLAS_mult_mean, colour="BLAS_mult"), data = df)
    }
}

    if (identical(df,size_and_better_mult)){

Treci zada:

Za pomoc jzyka R przeanalizowa dowolne dane z poprzednich laboratoriw. Elementy obowizkowe:

  1. W jednej tabeli jzyka R naley umieci wyniki dwch rnych eksperymentw (t.j. czas dziaania dwch rnych funkcji) dla rnych parametrw (t.j. rozmiarw wektora(macierzy)) (1p.)
  2. Tabela powinna zawiera dane z 10-krotnego uruchomienia tego samego eksperymentu dla tych samych parametrw (dla kadego takiego zestawu)
  3. Naley przedstawi wykresy rednich czasw obliczenia wybranych funkcji w zalenoci od rozmiaru wektora (macierzy) (1p.)
  4. Wykresy powinny zawiera informacj o odchyleniu standardowym dla uzyskanych wynikw. Supki bdw mona narysowa korzystajc z funkcji geom_errorbar pakietu ggplot2 (1p.)
  5. Uy aproksymacji wielomianowej dostpnej w jzyku R do znalezienia odpowiednich wielomianw, ktre najlepiej pasuj do wynikw kadego z algorytmw. Doda wykresy uzyskanych wielomianow do wykresu. (1p.)
  6. Dopasowa krzyw wykadnicz do liczby zakaonych koronawirusem. Przykadowe rdo danych w GoogleDoc lub https://ourworldindata.org/explorers/coronavirus-data-explorer. Jak zmieniaj si krzywe w zalenoci od wybranego okresu czasu. Porwna na wykresach co najmniej 2 dopasowania (2p.)
