Zadanie A¶

Metoda wykorzystująca przedział i zmianę znaku funkcji - Metoda Bisekcji¶

Opiera si ona na twierdzeniu Darboux: Jeeli funkcja ciga f(x) ma na kocach przedziau domknitego wartoci rnych znakw, to wewntrz tego przedziau, istnieje co najmniej jeden pierwiastek rwnania f(x) = 0.

Zaoenia:

  1. Funkcja f(x) jest ciga w przedziale domknitym [a, b]
  2. Funkcja przyjmuje rne znaki na kocach przedziau: f(a)*f(b) < 0

Algorytm polega na poowieniu przedziaw, ktrych znaki kracw si rni - gwarantuje to znalezienie pierwiastka.

Gwarantuje ona znalezienie jednego pierwiastka funkcji speniajcej zaoenia. Jej rzd zbienoci jest liniowy, jest zatem wolno zbiena.

Metoda wykorzystująca pochodną - Metoda Newtona-Raphsona¶

W metodzie tej przyjmuje si zaoenia:

  • W przedziale [a, b] znajduje si dokadnie jeden pierwiastek.
  • Funkcja ma rne znaki na kracach przedziau, tj. $f(a) * f (b) < 0$.
  • Pierwsza i druga pochodna funkcji maj stay znak w tym przedziale.

Algorytm polega na wybraniu punktu pocztkowego $x_{1}$ (initail guess, zazwyczaj jest to warto a, b 0 lub 1), a nastpnie iteracyjnego wyznaczania kolejnych przyblie pierwiastka funkcji wedug wzoru:

$x_{k+1} = x_{k} - \frac{f(x_{k})}{f(x_{k})}$<\center>

Kolejne przyblienia pierwiastka funkcji s przeciciami stycznej do funkcji w punkcie poprzednim z osi OX.

Metoda Steffensen'a¶

Minusem metody Newtona-Raphsona jest konieczno znajomoci jawnego wzoru pochodnej rozpatrywanej funkcji. Chcc tego unikn, a jednoczenie zachowujc wiksz szybko ni w metodzie bisekcji, korzystne jest zastosowanie metody Steffensen'a.

Metoda ta czy w sobie ide metody siecznych, nalecej do klasy metod quasi-Newtonowskich (przyblianie pochodnej ilorazem rnicowym), z ide akceleracji Aitkena. Korzysta si z niej z przewidywania, e przy rosncym i:

$x_{i+1} - x_{i} = f(x_{i})$<\center>

wtedy kolejne ilorazy rnicowe moemy zapisa jako:

$\frac{f(x_{i} + f(x_{i})) - f(x_{i})}{f(x_{i})}$<\center>

ostatecznie otrzymujemy:

$x_{i+1} = x_{i} - \frac{(f(x_{i}))^2}{f(x_{i} + f(x_{i})) - f(x_{i})}$<\center>

Rzd zbienoci tej metody wynosi 2, jest zatem lepszy od metody siecznych. Jednak kady krok wymaga policzenia dwch nowych wartoci funkcji, za w metodzie siecznych poczwszy od drugiego kroku liczymy tylko jedn now warto.

function getOutputOfFindZero(arg1, arg2, arg3)
    output = ""

        rd, = redirect_stdout()

            x = find_zero(arg1, arg2, arg3, verbose=true)

            x = "-1" # Error handling
        finally
            redirect_stdout(old_stdout) # Back to original stdout
        end
        output = String(readavailable(rd))
    
    end
    return x, output
end

Zadanie B¶

Wstęga Newtona¶

Wstga Newtona to fraktal tworzony na paszczynie liczb zespolonych. Powstaje on przez kolejne iteracje metody Newtona stosowanej na pewnej funkcji o N pierwiastkach zespolonych. Kady z N pierwiastkw stanowi punkt startowy dla iteracji Newtona. Jeli cig puntkw generowanych iteracyjnie zbiega do jednego z N pierwiastkw, cig ten kolorowany jest swoistym dla tego pierwiastka kolorem. Jednak kady wielomian stopnia co najmniej drugiego, podlegajcy iteracji Newtona, posiada punkty niezbiegajce do adnego z pierwiastkw, wyznaczany jest dla nich kolor N+1.
